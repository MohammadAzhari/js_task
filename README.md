## Run Locally

Clone the project

```bash
  git clone https://github.com/mohammadazhari/js_task
```

Go to the project directory

```bash
  cd js_task/
```

Install dependencies

```bash
  cd server/
  npm install
```

```bash
    cd ../
  cd client/
  npm install
```

Start the server

```bash
  cd server/
  npm start
```

Test the server

```bash
  npm test
```

Start the Client

`open now terminal`

```bash
  cd client/
  npm start
```
